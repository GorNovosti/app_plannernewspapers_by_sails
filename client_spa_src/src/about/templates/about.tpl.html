Общее описание

Веб-приложение "Планировщик выпуска(номеров) газет"(demo версия) предназначено для планирования содержания выпусков(номеров) газет с использованием графического интерфейса и работы в многопользовательском режиме.

Функциональные возможности

  1. Централизованное хранение данных
  2. Доступ к данным системы только авторизованными пользователям
  3. Создание выпусков(номеров) газет
  4. Создание шаблонов информационных блоков для ускорения процесса формирования содержания газетных страниц(полос)
  5. Цветовое оформление шаблонов информационных блоков

Системные требования для доступа к системе

>32.0
